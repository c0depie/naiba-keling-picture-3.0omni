# 可灵图片 3.0 Omni 示例提示词库

## 文生图示例

### 示例1：奇幻场景

```
奇幻丛林里，神女正伸手去够枝头上一颗饱满的红桃，神情专注又带着好奇。俯视拍摄，近景景别，浅景深虚化背景。利用层层叠叠的桃枝枝叶形成框式构图，焦点精确锁定在红桃与神女的手部。光线是柔和的散射自然光，在叶片间形成斑驳光影。画面带有真实电影颗粒感，真实电影质感。
```

**关键要素分析**:
- 主体: 神女
- 动作: 伸手够红桃
- 场景: 奇幻丛林、桃林
- 角度: 俯视拍摄
- 景别: 近景
- 构图: 框式构图
- 光线: 散射自然光、斑驳光影
- 质感: 真实电影颗粒感

### 示例2：都市夜景

```
繁华都市的夜晚，霓虹灯光在湿漉漉的街道上反射。一位穿黑色风衣的男人独自走在人行道上，背影。平视拍摄，中景景别。低角度的霓虹灯光形成彩色光斑，远处高楼大厦的灯光虚化成光晕。冷色调为主的青蓝色调，带有橙红色的霓虹灯光点缀。赛博朋克氛围，真实电影质感。
```

**关键要素分析**:
- 主体: 穿黑色风衣的男人
- 动作: 独自行走、背影
- 场景: 繁华都市夜晚、湿漉漉的街道
- 角度: 平视
- 景别: 中景
- 光线: 低角度霓虹灯光、彩色光斑
- 色调: 青蓝主调、橙红点缀
- 风格: 赛博朋克

### 示例3：古风武侠

```
竹林深处，一位白衣侠客拔剑出鞘，剑光如水。近景景别，平视拍摄。竹叶飘落的瞬间，动态模糊。阳光穿过竹林形成丁达尔效应的光柱。背景竹林虚化，前景竹叶清晰。水墨画风格与真实电影质感结合。青绿色调为主，剑光泛着冷银色。武侠氛围，凝重又带着决绝。
```

**关键要素分析**:
- 主体: 白衣侠客
- 动作: 拔剑出鞘
- 场景: 竹林深处
- 景别: 近景
- 光线: 阳光、丁达尔效应
- 风格: 水墨画+真实电影
- 色调: 青绿主调、冷银剑光

### 示例4：科幻场景

```
太空站内部，零重力环境下，一名宇航员漂浮着。舱内是银灰色金属墙壁，各种仪表盘闪烁着蓝绿色指示灯。透过巨大的圆形舷窗，可以看到蔚蓝的地球。广角镜头，全景景别。柔和的内部照明与窗外明亮的地球光形成对比。真实电影质感，冷色调为主的蓝银色系。科技感与孤独感并存。
```

## 系列组图示例

### 示例1：故事板序列（4张）

```
【第1张 - 远景开场】
荒漠边缘，夕阳西下。一座孤零零的驿站。远景景别，广角镜头。沙漠的金黄色与天空的橙红色渐变。低角度阳光形成长影。真实电影质感。

【第2张 - 中景人物】
驿站内，一位侠客坐在桌边喝酒。中景景别，平视。昏暗的烛光照明，前景酒杯清晰，背景虚化。暖色调为主的橙黄色。

【第3张 - 特写动作】
侠客的手握住剑柄，青筋暴起。特写镜头，浅景深。烛光在手部形成高光，暗部细节丰富。紧张氛围。

【第4张 - 远景结尾】
驿站外，数名黑衣人骑马而来，扬起尘土。远景景别，仰视角度。逆光剪影效果。戏剧感强烈。
```

### 示例2：角色多角度（3张）

```
【第1张 - 正面肖像】
年轻女性的正面肖像，面带微笑。近景景别，平视。柔和的室内光线，背景简洁。人像摄影风格，肤色自然，色调清新明亮。

【第2张 - 侧面轮廓】
同一女性的侧面轮廓，展示面部线条。近景景别，侧光照明形成轮廓光。黑色背景，突出主体。低调风格，黑白或深灰色调。

【第3张 - 全身姿态】
同一女性的全身站立姿态，穿着白色连衣裙。全景景别，平视。自然光照明，背景是公园绿地。清新自然风格，色调明亮柔和。
```
