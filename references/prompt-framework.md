## 第1层：基础信息 (必须)

### 组成要素
| 要素 | 说明 | 示例 |
|------|------|------|
| 主体 | 图片的核心对象 | 一个穿着红色连衣裙的年轻女性 |
| 动作 | 主体在做什么 | 正伸手去够枝头的红桃 |
| 场景 | 环境描述 | 茂密的桃林，层层叠叠的绿叶 |

### 写作技巧
- **具体化**: 不说"一个人"，说"一个穿白衬衫的男人"
- **动态化**: 不说"站着"，说"微微侧身，目光凝视"
- **环境化**: 不说"室内"，说"昏暗的地铁车厢"

## 第2层：摄影专业 (推荐)

### 2.1 景别 (Shot Size)

| 景别 | 适用场景 | 示例描述 |
|------|----------|----------|
| 远景 (Extreme Wide) | 环境展示 | 辽阔的草原，远山连绵 |
| 全景 (Wide/Full) | 全身展示 | 人物全身，展现与环境关系 |
| 中景 (Medium) | 半身互动 | 人物腰部以上，对话场景 |
| 近景 (Medium Close) | 面部表情 | 人物胸部以上，情绪表达 |
| 特写 (Close-up) | 细节强调 | 手部动作、面部细节 |
| 大特写 (Extreme Close) | 极致细节 | 眼睛、物品特写 |

### 2.2 拍摄角度 (Camera Angle)

| 角度 | 效果 | 适用 |
|------|------|------|
| 平视 | 自然、客观 | 大多数场景 |
| 俯视 | 压迫、渺小 | 展示环境、弱化角色 |
| 仰视 | 威严、高大 | 英雄、建筑 |
| 鸟瞰 (Bird's Eye) | 全局视角 | 城市俯瞰、战场 |
| 荷兰角 (Dutch Angle) | 不安、紧张 | 惊悚、悬疑 |

### 2.3 镜头类型 (Lens Type)

| 类型 | 说明 | 示例 |
|------|------|------|
| 过肩镜头 | 展示对话关系 | 以前景人物肩膀为框架 |
| 主观镜头 (POV) | 角色视角 | 模拟角色所见 |
| 跟镜头 | 跟随运动 | 跟随角色移动 |
| 推拉镜头 | 情感变化 | 推近强调，拉远展示 |

### 2.4 焦距与光圈

| 焦距 | 效果 | 光圈 | 效果 |
|------|------|------|------|
| 广角 | 夸大透视、容纳更多 | 大光圈 | 浅景深、背景虚化 |
| 标准 | 人眼视觉、自然 | 小光圈 | 深景深、前后清晰 |
| 长焦 | 压缩空间、主体突出 | - | - |

## 第3层：构图与光影 (进阶)

### 3.1 构图类型

| 构图法 | 说明 | 适用场景 |
|--------|------|----------|
| 三分法 | 主体放在三分线 | 通用场景 |
| 黄金分割 | 更自然的比例 | 人像、风景 |
| 对称构图 | 中心对称 | 建筑、倒影 |
| 引导线 | 线条引导视线 | 道路、河流 |
| 框架式 | 前景形成框架 | 门窗、树枝 |
| 对角线 | 动态感 | 运动、冲突 |

### 3.2 光线方向

| 光线 | 效果 | 适用 |
|------|------|------|
| 自然光 | 自然、柔和 | 日景、户外 |
| 侧光 | 立体感强 | 质感展示 |
| 逆光 | 轮廓、剪影 | 诗意、戏剧 |
| 顶光 | 压抑、激烈 | 正午、审讯 |
| 底光 | 恐怖、诡异 | 惊悚、恐怖 |

### 3.3 光影效果

| 效果 | 描述 |
|------|------|
| 斑驳光影 | 光线透过遮挡形成的光斑 |
| 轮廓光 | 主体边缘的光线 |
| 伦勃朗光 | 人像经典布光 |
| 散射光 | 柔和无方向 |

## 第4层：风格与质感 (专业)

### 4.1 质感类型

| 质感 | 描述 | 适用 |
|------|------|------|
| 真实电影质感 | 专业电影拍摄效果 | 电影感场景 |
| 胶片质感 | 传统胶片颗粒和色彩 | 怀旧、艺术 |
| 数字质感 | 清晰锐利 | 现代、科技 |
| 画面颗粒感细腻 | 高质量电影胶片 | 高级感 |

### 4.2 胶片类型

| 胶片 | 特点 | 色调倾向 |
|------|------|----------|
| 柯达 Vision | 电影标准 | 饱和、暖色 |
| 柯达 Portra | 人像胶片 | 自然肤色 |
| 富士 Velvia | 幻灯片 | 高饱和、鲜艳 |
| 宝丽来 | 即拍即得 | 复古、偏色 |

### 4.3 色调方案

| 色调 | 效果 | 示例 |
|------|------|------|
| 青灰/墨黑 | 冷峻、紧张 | 惊悚、犯罪 |
| 暖色调 | 温暖、怀旧 | 回忆、温馨 |
| 冷色调 | 冷静、疏离 | 科幻、未来 |
| 高对比 | 强烈、冲击 | 黑色电影 |
| 低对比 | 柔和、梦幻 | 文艺、回忆 |

## 第5层：氛围与情绪 (点睛)

### 5.1 整体氛围

| 氛围 | 关键词 |
|------|--------|
| 紧张 | 戏剧感、冲突、压迫 |
| 宁静 | 和谐、安详、平和 |
| 梦幻 | 柔和、虚幻、诗意 |
| 神秘 | 隐约、不可知 |
| 欢快 | 明亮、活力、动感 |

### 5.2 表情细节

| 情绪 | 描述方式 |
|------|----------|
| 专注 | 眼神聚焦、神情专注 |
| 好奇 | 探索、带着疑问 |
| 紧张 | 肌肉紧绷、警惕 |
| 温柔 | 面带微笑、眼神柔和 |

### 5.3 画面感

| 感受 | 关键词 |
|------|--------|
| 冲击力 | 强烈、震撼、视觉冲击 |
| 诗意 | 意境、韵味、含蓄 |
| 节奏感 | 动静结合、张弛有度 |
