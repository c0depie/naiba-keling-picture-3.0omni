# 可灵图片 3.0 Omni 影视术语词典

## 景别术语

| 中文 | 英文 | 说明 |
|------|------|------|
| 远景 | Extreme Wide Shot (EWS) | 展示广阔环境，人物较小 |
| 全景 | Wide Shot (WS) / Full Shot | 人物全身，展现与环境关系 |
| 中景 | Medium Shot (MS) | 人物腰部以上 |
| 近景 | Medium Close-Up (MCU) | 人物胸部以上 |
| 特写 | Close-Up (CU) | 面部或细节 |
| 大特写 | Extreme Close-Up (ECU) | 极致细节，如眼睛、嘴唇 |

## 拍摄角度

| 中文 | 英文 | 效果 |
|------|------|------|
| 平视 | Eye Level | 自然、客观，最常用 |
| 俯视 | High Angle | 压迫感，主体显得渺小 |
| 仰视 | Low Angle | 威严、高大、英雄感 |
| 鸟瞰 | Bird's Eye View | 上帝视角，全局展示 |
| 荷兰角 | Dutch Angle | 倾斜构图，不安、紧张 |
| 虫视 | Worm's Eye View | 极低角度仰拍 |

## 镜头类型

| 中文 | 英文 | 说明 |
|------|------|------|
| 过肩镜头 | Over-the-Shoulder (OTS) | 以前景人物肩膀为框架，展示对话 |
| 主观镜头 | Point of View (POV) | 角色主观视角 |
| 跟镜头 | Tracking Shot | 跟随主体移动 |
| 推镜头 | Dolly In | 镜头向主体推进，强调情感 |
| 拉镜头 | Dolly Out | 镜头远离主体，展示环境 |
| 摇镜头 | Pan/Tilt | 水平或垂直转动镜头 |
| 悬停镜头 | Hovering Shot | 悬空观察效果 |

## 焦距术语

| 中文 | 焦距范围 | 效果 |
|------|----------|------|
| 超广角 | 14-24mm | 夸张透视，视觉冲击 |
| 广角 | 24-35mm | 包含环境，轻微变形 |
| 标准 | 35-70mm | 人眼视觉，自然 |
| 中长焦 | 70-135mm | 轻微压缩，人像常用 |
| 长焦 | 135mm+ | 空间压缩，背景虚化 |

## 光圈与景深

| 中文 | 说明 |
|------|------|
| 大光圈 (f/1.2-f/2.8) | 浅景深，背景虚化 |
| 小光圈 (f/8-f/16) | 深景深，前后清晰 |
| 浅景深 | 主体清晰，背景虚化 |
| 深景深 | 前后都清晰 |
| 焦外虚化 | 背景光斑效果 |
| 焦点锁定 | 焦点精确在主体上 |

## 构图类型

| 中文 | 英文 | 说明 |
|------|------|------|
| 三分法 | Rule of Thirds | 主体放在三分线上 |
| 黄金分割 | Golden Ratio | 更自然的比例 |
| 对称构图 | Symmetry | 中心对称，稳定感 |
| 引导线 | Leading Lines | 线条引导视线 |
| 框架式 | Framing | 前景形成框架 |
| 对角线 | Diagonal | 对角线构图，动态感 |
| 三角形 | Triangle | 稳定或倒三角不稳定 |
| S形曲线 | S-Curve | 柔和、流动感 |
